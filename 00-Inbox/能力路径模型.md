# 🧠 阶段2：抽象驱动（关键分水岭）

当你完成 GPIO + UART + Timer 后：

---

## 👉 你要做一件非常关键的事：

> ❗自己封装 Driver（不要直接用 HAL）

---

### 举例：

```c
void led_on();
void led_off();
```

而不是：

```c

```

---

## 🎯 为什么？

因为：

> **驱动 = 硬件 → 软件的抽象边界**

---

## 📌 你会获得：

- 模块化思维
    
- 硬件解耦能力
    
- 可复用性
