# 硬件进阶路线

> 类型：参考资料 | 用途：完成 Zephyr 基础后的下一步方向

## 路线一：芯片设计 — Tiny Tapeout

以极低成本（约 $100）将自己的数字逻辑设计印在真实硅片上。

- 学习路径：逻辑门 → Verilog → ASIC 流程
- 起点：[tinytapeout.com](https://tinytapeout.com)
- 进阶目标：设计一个硬件加速算子

## 路线二：边缘 AI — NPU 加速推理

把大模型推理搬到嵌入式硬件，调用 NPU 而不是只用 CPU。

- 推荐平台：RK3588（Orange Pi 5）/ Milk-V Duo（RISC-V）
- 核心技能：模型量化（RKNN/ONNX）、异构计算（CPU + GPU + NPU）
- 参考项目：LocalAI、EZ-WifiBroadcast

## 路线三：运动控制 — FOC 无刷电机

让硬件动得像丝绸一样顺滑，机械臂 / 四足机器人的核心技术。

- 核心知识：SVPWM、PID 闭环、编码器反馈
- 起点项目：[SimpleFOC](https://github.com/simplefoc/Arduino-FOC)
- 进阶目标：自制集成伺服驱动板

## 路线四：复杂系统 — 软硬结合

| 方向 | 代表项目 | 核心挑战 |
|------|---------|---------|
| 纯硬件 | PinePhone / Framework | 多层 PCB、高速信号完整性 |
| 软硬结合 | OpenWrt / Home Assistant | 高并发网络 I/O、嵌入式 Docker |
| 前沿 | RISC-V 软核 | 在 FPGA 上跑自己写的 CPU |
