# STM32 知识图谱

> 关联概念：[[Zephyr]] | 长期领域：嵌入式系统

## 一、建模目标

```
STM32知识 = 外设机制 + 数据流 + 控制流 + 时序
```

核心问题：**数据是怎么在芯片里流动的？谁在控制？什么时候发生？**

## 四、问题驱动清单（05-Questions 应建立）

**DMA 相关**
- 为什么 DMA 比中断快？
- DMA 什么时候触发？
- DMA 会不会占用总线？

**USART 相关**
- USART 为什么会丢数据？
- RXNE 中断什么时候触发？

**中断系统**
- 中断是怎么打断 CPU 的？
- NVIC 在干什么？

**系统级问题（最关键）**
- 数据从 USART 到内存经历了什么？
- CPU、DMA、总线三者关系是什么？
