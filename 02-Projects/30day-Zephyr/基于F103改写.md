# 二、你这四个板子分别代表什么

### 1 ❌ `gd32vf103c_starter / gd32vf103v_eval`

- 这是 **GD32VF103**
    
- RISC-V 架构
    
- ❗不是 STM32
    

👉 直接忽略

---

### 2 ✅ `nucleo_f103rb`

- 芯片：STM32F103RB（64引脚）
    
- 官方板子
    
- Zephyr 支持最完整
    

👉 **最推荐用来“借壳”**

---

### 3 ✅ `stm32f103_mini`

- 通常对应“最小系统板”
    
- 接近 Bluepill / 自制板
    

👉 **最适合你改成 ZE**

---

# 三、关键问题：ZE vs RB 差在哪？

你的芯片是：

👉 STM32F103ZE（144引脚）

对比：

|型号|Flash|RAM|引脚|
|---|---|---|---|
|F103RB|128KB|20KB|64|
|F103ZE|512KB|64KB|144|

👉 本质区别：

```text
资源更多 + IO更多
```

👉 核心（Cortex-M3）完全一样

👉 **Zephyr 的 SoC 层已经完全支持你**

你缺的只是：

```text
Board 描述（引脚定义）
```

所以：
